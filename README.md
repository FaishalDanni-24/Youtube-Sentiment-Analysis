# Youtube-Sentiment-Analysis
Model for Youtube comments sentiment analysis using Multinomial Naive Bayes.<br>
Preprocessing is done using LabelEncoder and TF-IDF Vectorizer.

# Files
- EDA.py — Code for EDA.
- Model.py — Code for modelling.
- README.md — Description.
